# CSS Restoration Progress Log

## Patterns (add reusable patterns here as you discover them)

### Tailwind + CSS Custom Property Pattern
Use `bg-[var(--token)]` for all colors. Never use Tailwind's built-in color classes (bg-blue-500, etc.).
```
className="bg-[var(--off-white)] dark:bg-[var(--surface-1)]"
className="text-[var(--taupe-5)] dark:text-[var(--taupe-1)]"
className="border-[var(--taupe-2)] dark:border-[var(--taupe-3)]"
```

### Font Family Pattern
```
className="font-[family-name:var(--pixel)]"   // ChicagoFLF — headings, labels
className="font-[family-name:var(--mono)]"     // IBM Plex Mono — data, metadata
className="font-[family-name:var(--sans)]"     // DM Sans — body text
```

### Bevel Pattern (defined in app.css @layer components)
```
className="bevel"         // raised 3D border
className="bevel-inset"   // pressed/inset 3D border
```

### Dark Mode Pattern
Always pair light and dark variants:
```
className="bg-[var(--off-white)] dark:bg-[var(--surface-1)] border-[var(--taupe-2)] dark:border-[var(--taupe-3)]"
```

### Alpha Color Pattern
For transparent tints, use RGB triplet tokens:
```
className="bg-[rgba(var(--violet-3-rgb),0.06)]"
className="dark:bg-[rgba(var(--violet-3-rgb),0.12)]"
```
